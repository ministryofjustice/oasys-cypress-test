checkLayer3Menu

  • checkLayer3Menu(sanMode: boolean): void

  • Checks the floating menu to see if sections 2 to 13 and the self-assessment form are there or not, and checks for the SAN and SP sections. Parameter is true for SAN mode, false for normal OASys mode (layer 3.1), the test fails if the menu is not as expected.

    Parameters

    • sanMode: boolean

    Returns void

      san.ts
