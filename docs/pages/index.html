Pages

oasys.Pages....page

The pages library contains a class definition of each page in the OASys application.

Pages need to be created as objects from the classes provided, for example:

const page = new pages.SentencePlan.ispSection52to8().goto().

Auto-complete drop-downs will show you the available groups and pages.

Click on the links on the left for more information about pages available, and see the Page class for a list of methods available for interacting with pages.
